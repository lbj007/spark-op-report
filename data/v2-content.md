# Spark思派克素养中心运营计划（OP）V2.0

> 本版本在V1基础上，针对北京小学数学线下门店实操场景进行了深度迭代优化。
> 核心改动：校准财务模型、补充选址策略、统一战略节奏、细化执行SOP。

---

## 整体概览

### 战略一句话总结

以北京单店为验证样本，通过"小班精品+校内提分+竞赛出口"的差异化定位，用18-24个月跑通单店盈利模型，北京稳步复制至5-8家校区，验证成功后再审慎布局新一线城市，最终实现标准化输出与规模化扩张。

### 三年战略里程碑（修正版）

| 阶段 | 时间 | 核心目标 | 关键指标 |
|------|------|----------|----------|
| 验证期 | Y1 | 北京单店盈利模型跑通 | 在读150-180人，续费率≥75%，月度盈亏平衡 |
| 复制验证期 | Y2 | 北京复制2-3家校区（分批开设） | 新店12个月内盈亏平衡，店长内部培养率≥70% |
| 稳健扩张期 | Y3 | 北京5-8家+试点1个新城市2家 | 根据Y2验证结果决定是否跨城 |

> ⚠️ **战略纪律：** Y2开设新店的前提条件是Y1单店连续3个月盈利；Y3跨城的前提条件是Y2所有门店6个月内实现盈亏平衡。任何一个前提未达成，就延后执行，绝不冒进。

### 战略成功关键要素（KSF）

```
┌──────────────────┐
│    教学质量       │ ← 续费率≥75%的根基
│  （教师质量）     │
└────────┬─────────┘
         │
    ┌────┼────┐
    │    │    │
    ▼    ▼    ▼
  获客  转化  续费
  效率   率    率
    │    │    │
    └────┼────┘
         │
    ┌────┴─────────┐
    │  单店盈利模型  │ ← 复制前提（18-24个月验证）
    └────┬─────────┘
         │
    ┌────┴─────────┐
    │  人才储备充足  │ ← 扩张前提（每店1名合格店长）
    └──────────────┘
```

**核心原则：稳中求进。** 教培行业的竞争是长跑，不是短跑。宁可慢一点，也要稳一点。

## 一、外部调研

> **洞察：** 少儿数学思维赛道正经历"供给侧改革"——传统学科机构退出、素养机构跑马圈地，窗口期就在今明两年。

### 1.1 行业调研

#### 市场规模与趋势

| 指标 | 数据 | 说明 |
|------|------|------|
| 2025年非学科培训市场规模 | 约4636.6亿元 | 含人文美育、科学思维、语言表达、编程四大品类 |
| 思维品类市场规模 | 约1140亿元 | 参照数学培训渗透率28%，22-25年CAGR=25% |
| 线下非学科机构数量 | K9阶段8.4万个 | 学前阶段4.5万个，存量机构出清中 |
| 头部机构线下扩张 | 好未来220+，新东方+20% | 头部借机收割市场，线下布局加速 |

**市场格局变化：**
- **存量洗牌：** 双减后微型机构无产品能力退出，小型机构无力扩张，中型机构观望——腾出市场空间
- **增量入场：** 头部机构加快线下布局，学而思素养220+学习中心，新东方素养加速渗透
- **需求迁移：** 家长从"学科提分"转向"能力培养"，数学思维成为最大公约数

#### 宏观经济环境分析

| 指标 | 数据 | 教育消费洞察 |
|------|------|-------------|
| 2024年3月二手住宅均价 | 回落到21年3月水平 | 家庭资产缩水，教育投入更趋理性 |
| 2024年5月新增房贷 | 514亿元（vs 2020年5月4662亿） | 购房需求暴跌，消费信心转移 |
| 2023年人均教育文化娱乐消费 | 2904元，增长17.6% | 教育支出增速第一，家长愿意为"效果"付费 |

**关键洞察：** 家长消费更理性了，但为"确定性效果"付费的意愿更强——这正是"校内提分"定位的机会点。

#### 学而思素养

**优势：** 品牌背书强（好未来上市公司）、产品体系全面、智能硬件辅助、获客能力强

**薄弱环节：** 课程难度偏高（含初中内容）、课堂节奏快压力大、班型大（15-20人）老师难以关注每个孩子

#### 新东方素养

**优势：** 30年教育IP、全国300+分店、名师体系、属地化运营

**薄弱环节：** 数学产品线弱（北京只有线上小班）、线下以语言素质类为主

#### 高思数学

**优势：** 竞赛体系成熟、海淀本地化运营、升学资源丰富

**薄弱环节：** 定位精英化（只适合极少数学霸）、普适性不足、全国品牌力弱

### 1.3 消费者画像

#### 目标客群特征

| 维度 | 特征 | 运营启示 |
|------|------|---------|
| 学员年龄 | 小学1-6年级（6-12岁） | 决策周期短，见效快 |
| 家长年龄 | 30-39岁 | 中产主力，信息敏感 |
| 决策人 | 女性为主（妈妈≥70%） | 沟通触点：妈妈优先 |
| 收入 | 中高收入家庭 | 愿意为品质付费 |
| 学历 | 本科及以上 | 重视理念和数据 |
| 关注点 | 成绩/能力、升学规划 | 痛点：校内成绩+择校焦虑 |

#### 家长决策旅程

```
认知 → 兴趣 → 考量 → 决策 → 体验 → 续费/转介绍
 │      │      │      │      │      │
 ▼      ▼      ▼      ▼      ▼      ▼
朋友   体验课  对比3家  优惠   效果   口碑
推荐   吸引    机构    促单   承诺   裂变
```

#### 影响决策的关键因素

1. **产品效果（40%）：** 能否提分？能否解决我的焦虑？
2. **位置便利（25%）：** 2公里内，停车方便，接送成本
3. **服务体验（20%）：** 老师专业吗？反馈及时吗？
4. **价格敏感（15%）：** 性价比 vs 品质

**运营启示：** 与其拼价格，不如拼"效果可视化"——让家长看得见提分，看得见进步。

## 二、选址策略（新增章节）

> **洞察：** 对线下门店来说，选址决定了60%的成败。错误的选址无法靠运营弥补。

### 2.1 选址核心标准

| 维度 | 标准 | 说明 |
|------|------|------|
| 位置 | 距目标小学800米以内 | 在家长接送动线上，方便放学后直接到店 |
| 楼层 | 1-3层 | 合规要求；优选2层（租金性价比最优） |
| 面积 | 200-350㎡ | 4-6间教室+前台+家长等候区+办公区 |
| 租金 | ≤7.5元/㎡/天 | 月租金控制在月营收15%以内 |
| 周边学校 | 半径1.5公里内≥3所小学 | 在校生总数≥3000人 |
| 竞品密度 | 同商圈学而思/高思≤2家 | 避免正面硬刚头部 |
| 交通配套 | 有停车位或公交/地铁便利 | 家长接送便利性直接影响续费 |
| 物业类型 | 商业/综合体底商 | 避免纯住宅底商（人流不足） |

### 2.2 选址排除项

- ❌ 4层以上（消防合规不通过）
- ❌ 纯住宅区底商（人流量不足，且邻居投诉风险）
- ❌ 与学科培训机构同一物业（互相举报风险）
- ❌ 周边1公里内无小学（生源不足）
- ❌ 租金超过8元/㎡/天（成本不可控）
- ❌ 物业不允许做教育培训的场地

### 2.3 选址评估打分表

| 评估项 | 权重 | 评分标准（1-5分） |
|--------|------|-----------------|
| 目标小学距离 | 25% | 5=300米内，4=500米内，3=800米内，2=1公里内，1=1公里外 |
| 租金水平 | 20% | 5=≤5元/㎡/天，4=≤6元，3=≤7元，2=≤8元，1=>8元 |
| 交通便利性 | 15% | 5=地铁口+停车场，4=地铁500米内，3=公交便利，2=一般，1=不便 |
| 楼层与面积 | 15% | 5=1-2层+250-350㎡，4=3层，3=面积偏小，2=面积偏大，1=不合规 |
| 竞品密度 | 10% | 5=无直接竞品，4=1家，3=2家，2=3家，1=饱和 |
| 周边生源量 | 10% | 5=5000+人，4=3000+人，3=2000+人，2=1000+人，1=<1000人 |
| 物业条件 | 5% | 5=全新装修，4=简单翻新可用，3=需中等装修，2=需大改，1=不合规 |

**总分≥4.0方可签约，3.5-4.0需校长+运营总监共同评估，<3.5放弃。**

### 2.4 北京优选区域建议

| 优先级 | 区域 | 理由 |
|--------|------|------|
| 第一梯队 | 海淀（万柳/上地/五道口） | 教育氛围最浓，家长付费意愿最强 |
| 第一梯队 | 西城（金融街/德胜门） | 优质小学密集，中产家庭集中 |
| 第二梯队 | 朝阳（望京/朝阳公园） | 新兴中产聚集区，竞争相对缓和 |
| 第二梯队 | 丰台（总部基地/丰台科技园） | 新小区多，生源增长快，租金性价比高 |
| 第三梯队 | 昌平（回龙观/天通苑） | 人口密度大但客单价敏感，适合Y2验证 |

### 2.5 校区空间规划

```
【标准校区 250㎡ 空间布局】

┌──────────────────────────────────────────┐
│  前台接待区（15㎡）│   家长等候区（25㎡） │
│  形象墙+咨询台     │  沙发+展示+饮水机    │
├──────────┬─────────┴────────┬────────────┤
│ 教室A    │    教室B         │  教室C     │
│ 30㎡     │    30㎡          │  30㎡      │
│ 8-10人   │    8-10人        │  8-10人    │
├──────────┼──────────────────┼────────────┤
│ 教室D    │    教室E         │  多功能区   │
│ 30㎡     │    25㎡          │  20㎡      │
│ 8-10人   │    6-8人         │  测评/面谈  │
├──────────┴──────────────────┴────────────┤
│  办公区（20㎡）│ 储物间（10㎡）│ 卫生间（15㎡）│
└──────────────────────────────────────────┘
```

**空间设计原则：**
- 家长等候区必须能看到教室（玻璃隔断），增强信任感
- 每间教室配白板+投影+空调，面积≥25㎡
- 前台区域做品牌形象展示：学员成果墙、荣誉证书、课程体系图
- 消防通道不少于2个，灭火器/烟感器按标准配置

## 三、产品规划

> **洞察：** 产品是机构的命门。好产品要让家长"看得见效果"，让续费"顺理成章"。开业阶段不要三线并行，分阶段开课。

### 3.1 产品定位

| 课程类型 | 说明 | 目标年级 | 核心价值 |
|---------|------|---------|---------|
| 培优课（AP） | 提升数学思维，拓展解题能力 | 1-6年级 | 思维培养 |
| 同步课（新数学） | 校内同步数学，解决校内成绩问题 | 3-6年级 | 校内提分 |
| 竞赛课（CP） | 国际竞赛冲刺，积累竞赛成果 | 3-6年级 | 升学加分 |

### 3.2 分阶段开课策略（新增）

> ⚠️ 开业资源有限（3-4名教师），不要三线齐发，分阶段启动。

```
阶段一（开业月）：只开培优课
├── 年级：1-3年级
├── 目的：引流+验证教学+积累口碑
├── 教师：2-3名
└── 理由：低年级无考试压力，家长更包容，容错空间大

阶段二（开业后第3个月）：增开同步课
├── 年级：3-5年级
├── 目的：这是现金牛课程，直接解决家长"提分"焦虑
├── 教师：增至4名
└── 理由：有了第一批学员口碑，同步课更容易转化

阶段三（开业后第6个月）：小规模开竞赛课
├── 年级：3-6年级（从现有学员中筛选）
├── 目的：做口碑灯塔，用竞赛成绩驱动转介绍
├── 教师：1名竞赛专项教师（可外聘兼职）
└── 理由：竞赛课需要高水平教师，初期很难招到全职
```

### 3.3 年级入口策略与课程配比（优化版）

| 年级 | 主力课程 | 课程内容配比 | 卖点话术 |
|------|---------|-------------|---------|
| 1年级 | 培优课 | 30%计算基础+70%思维游戏 | "让孩子爱上数学，建立数感" |
| 2年级 | 培优课 | 35%计算技巧+65%思维拓展 | "思维启蒙黄金期，打好底子" |
| 3年级 | 同步课+培优课 | 60%校内+40%思维 | "三年级是分水岭，校内提分看得见" |
| 4年级 | 同步课+培优课 | 65%校内+35%思维 | "校内稳固+思维突破两不误" |
| 5年级 | 同步课+培优/竞赛 | 75%校内+25%竞赛衔接 | "冲刺小升初，目标校不是梦" |
| 6年级 | 同步课+竞赛课 | 80%校内总复习+20%竞赛冲刺 | "毕业冲刺+竞赛成果双丰收" |

### 3.4 产品金字塔

```
        ┌─────────────┐
        │   竞赛课     │ ← 塔尖（10%学员）
        │  AMC8/SASMO  │   做口碑灯塔
        └──────┬──────┘
       ┌───────┴───────┐
       │    培优课      │ ← 中坚（40%学员）
       │  思维拓展/应用  │   引流+留存
       └───────┬───────┘
    ┌──────────┴──────────┐
    │      同步课          │ ← 基座（50%学员）
    │   校内同步/基础巩固   │   现金牛
    └─────────────────────┘
```

**产品理念：** 新加坡数学 = 校内提分 + 思维培养
- 家长要的是"确定性"——校内提分最直接可见
- 纯思维课"见效慢"，家长没耐心
- 纯应试课"没特色"，不符合素养定位

### 3.5 班型阶梯设置

| 难度等级 | 说明 | 准入方式 | 适合学员 |
|---------|------|---------|---------|
| 基础班 | 夯实基础，提升兴趣 | 入学测评 | 校内成绩中等偏下 |
| 提升班 | 提升思维，拓展思路 | 阶段性测试+教师推荐 | 校内成绩中等偏上 |
| 竞赛班 | 专注竞赛，冲刺获奖 | 竞赛选拔+测评 | 校内成绩优秀 |

**开班规则：**

| 规则 | 标准 |
|------|------|

#### 获客渠道详解

**🥇 小红书（最推荐）**
- 精英妈妈聚集地，内容种草能力强
- 重点内容方向：海淀妈妈经验分享、思维启蒙干货、升学规划指南
- 执行建议：
  - 建立官方账号（每周3篇笔记）
  - 招募KOC（关键意见消费者）矩阵
  - 关键词：#数学思维 #海淀妈妈 #幼升小 #STEM

**🥇 转介绍（最高效）**
- 精英家长圈层效应强，一人推荐带动一圈
- 关键动作：服务好现有家长，让他们主动分享；设计灯塔家长计划
- 目标：转介绍占比达到35%+

**🥈 异业合作（低成本）**
- 优质合作伙伴：高端幼儿园/小学、美术馆/科技馆、高尔夫球会/马术俱乐部、儿童摄影/绘本馆
- 合作形式：联合活动、互相导流、资源置换

#### 销售链路

#### 销售转化SOP

**第一步：线索获取与筛选**

| 动作 | 时限 | 内容 | 负责人 |
|------|------|------|-------|
| 线索分配 | **30分钟内** | 根据年级、区域分配给咨询师 | 销售主管 |

| 意向分类 | 当天 | A（高意向）/B（考虑）/C（低意向） | 咨询师 |

> ⚠️ **关键提醒：** 30分钟内响应是转化关键，晚则流失50%

**第二步：预约上门**

| 动作 | 时限 | 内容 | 负责人 |
|------|------|------|-------|
| 预约排课 | 3天内 | 预约试听时间 | 咨询师 |
| 短信提醒 | 前1天 | 发送时间、地点、注意事项 | 咨询师 |
| 微信确认 | 当天上午 | 再次确认到店时间 | 咨询师 |

**第三步：试听转化（重点优化）**

| 动作 | 时限 | 内容 | 负责人 |
|------|------|------|-------|
| 接待签到 | 到店即时 | 填写基本信息、引导参观校区、饮品接待 | 前台/教务 |
| 水平测评 | 15分钟 | 了解孩子当前水平，为后续咨询提供数据 | 教师 |
| 试听课 | 45分钟 | **完整正课体验**（不是简化版） | 教师 |
| 家长旁听 | 同步 | 安排家长在等候区通过玻璃观察课堂 | 教务 |
| 咨询转化 | 20分钟 | 学情分析→方案推荐→异议处理→逼单 | 咨询师 |

**试听课45分钟标准流程（新增）：**

```
0-5分钟：破冰暖场（趣味数学小游戏，让孩子放松）
5-10分钟：知识导入（用实物/教具引入今天主题）
10-25分钟：核心教学（CPA教学法完整展示）
25-35分钟：互动练习（确保每个孩子都参与发言）
35-40分钟：成果展示（让孩子向家长展示今天学到的）
40-45分钟：总结鼓励（给每个孩子具体的正面反馈）
```

> **关键要求：** 试听课=正式课体验。不要搞简化版或表演课，家长一眼就能看穿。让家长看到的就是孩子未来每节课的真实体验。

**第四步：咨询转化20分钟话术框架（新增）**

```
1. 效果确认（3分钟）
   "您看到孩子刚才在课堂上的表现了吧？他/她在XX环节表现特别好……"
   → 让家长回忆刚才的正面体验

2. 问题诊断（5分钟）
   "根据测评结果，孩子在XX方面基础不错，但XX方面需要加强……"
   → 专业分析，建立信任

3. 方案推荐（5分钟）
   "针对孩子的情况，我建议先上XX课，每周2次……"
   → 给出具体方案，不是让家长自己选

4. 异议处理（5分钟）
   - "再考虑考虑" → "完全理解，您主要是在考虑哪方面？"
   - "有点贵" → "我们季度包135元一课时，比XX便宜，而且小班服务……"
   - "离家远" → "每周来2次，每次1.5小时，很多家长觉得值得"

5. 促成（2分钟）
   "这周还有X个名额，现在报名可以享受……"
   → 稀缺性+优惠促单
```

**未成交客户7天跟进节奏（新增）：**

### 8.2 收入测算模型（修正版）

#### 收入测算假设

| 参数 | 乐观场景 | 保守场景 |
|------|---------|---------|
| 课时单价 | 135元 | 125元 |
| 平均班型 | 8人 | 6人 |
| 周均开班 | 18班 | 12班 |
| 周均满班率 | 80% | 65% |
| 周上课次数 | 8次 | 6次 |

#### 收入测算

```
乐观场景（月收入）：
- 周末班：12班 × 8人 × 80% × 8次 × 135元 ≈ 8.3万/周 × 4 = 33万/月
- 周内班：6班 × 6人 × 70% × 6次 × 135元 ≈ 2万/周 × 4 = 8万/月
- 合计：约20-25万/月（成熟期）

保守场景（月收入）：
- 约8-12万/月（成长期）

实际爬坡预测：
- 第1-3月：月收入3-5万（在读30-50人）
- 第4-6月：月收入5-8万（在读60-80人）
- 第7-12月：月收入8-15万（在读100-150人）
- 第13-18月：月收入15-20万（在读150-200人）→ 盈亏平衡
- 第19-24月：月收入20-25万（在读200人+）→ 稳定盈利
```

### 8.3 资金需求全景图（新增，核心修正）

> ⚠️ **这是本次优化最重要的修正之一。** 原文暗示150万即可启动，但实际资金需求远高于此。

```
【单店资金需求总览】

一、开业前投入：140-175万
   ├── 押金+证照：25-35万
   ├── 装修+设备：37-50万
   ├── 教材+推广+培训：23-33万
   └── 流动资金：30-40万

二、开业后亏损期补贴（约12-15个月）：
   ├── 第1-6月：月均亏损10-15万 → 小计60-90万
   ├── 第7-12月：月均亏损3-8万 → 小计18-48万
   └── 小计：80-140万

三、安全冗余：30-50万

四、总计：250-365万（取中值约300万）

⚠️ 如果只准备了150万就开店，大概率在第8-10个月现金流断裂。
   请至少准备300万的可调配资金（含自有+可快速调用的借款额度）。
```

### 8.4 盈亏平衡分析

- 按135元定价（季度包）：约需200-220人在读才能盈亏平衡

- **结论：** 盈亏平衡至少需要200+人在读

## 九、风险与应对

> **洞察：** 风险管理的核心是"预防优于补救"。要把风险想在前面，建立预警机制，确保问题发生时能快速响应。

### 9.1 风险全景图

> ⚠️ **核心警示：** 扩张速度超出运营能力边界，是项目失败的最大概率场景。

```
┌─────────────────────────────────────────────────────┐
│          「扩张速度超出运营能力边界」风险推演          │
├─────────────────────────────────────────────────────┤
│                                                      │
│  核心矛盾：                                          │
│  小班精品（6-12人）极度依赖师资质量与管理效率         │

│                                                      │
│  风险结果：                                           │
│  单店模型尚未跑通就大规模扩张                         │

│                                                      │
│  防控原则：                                           │
│  ✅ Y1只做1家，18个月验证                             │
│  ✅ Y2分3批开设（1+1+1），每批间隔3-4个月            │

└─────────────────────────────────────────────────────┘
```

### 附录3：V1→V2核心改动一览

| 改动项 | V1原文 | V2优化 | 理由 |
|--------|-------|-------|------|
| 战略节奏 | Y1→Y2北京5家→Y3北上广深20家 | Y1→Y2北京2-3家→Y3北京5-8家+试1城 | 过于激进，前后矛盾 |
| 选址策略 | 缺失 | 新增完整章节 | 线下门店命门 |
| 品牌定位 | 精英家庭数理智选品牌 | 专注小学数学的小班精品课堂 | 原定位太虚，劝退中产 |
| 竞品话术 | 直接攻击竞品 | 对比引导话术，不提竞品名字 | 避免品牌风险 |
| 获客成本 | ≤1000元 | ≤1200元 | 原目标不符合北京实际 |
| 总资金需求 | 暗示150万 | 明确300-365万 | 原文严重低估亏损期补贴 |
| 定价策略 | 全年包125元/课时 | 季度包135元主推+分层定价 | 合规限制（≤3个月收费） |
| 新师考核 | 0-1年不设考核 | 渐进式考核，第6月正式挂钩 | 家长3个月看不到效果就退费 |
| 师生比 | 初期1:30（3人服务100人） | 初期1:25（4人服务80人） | 原计算未考虑实际排课率 |

| 同行举报 | 一句话带过 | 完整防举报SOP | 北京最现实的风险 |
| 火花品牌风险 | 未提及 | 新增风险及逐步独立策略 | 母品牌依赖风险 |
| 寒暑假空窗 | 未提及 | 新增提前续费策略 | 现金流断点风险 |
| 试听课流程 | 只有框架 | 45分钟完整剧本+咨询话术 | 执行层面最需要的内容 |
| 班主任SOP | 缺失 | 新增日/周/月工作清单 | 服务落地的关键 |

> **全文总结：** 这份OP的核心逻辑是对的——小班精品+校内提分+竞赛出口，这个定位在北京线下教培市场确实有空间。V2版本的核心改动是让数字更真实、让执行更落地、让风险更可控。记住三句话：**选址定生死、师资定口碑、现金流定生存。**
